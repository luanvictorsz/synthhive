# SynthHive — Landing Page

Studio de Audiovisual & Jogos. Stack: **React + Vite**.

## Instalação e execução

```bash
npm install
npm run dev
```

Acesse em http://localhost:5173

## Build para produção

```bash
npm run build
npm run preview
```
